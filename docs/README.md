## Importing to Notion

To import these docs into Notion:

1. **Method 1 - Copy/Paste:**
   - Open the .md file
   - Select all (Ctrl+A)
   - Copy (Ctrl+C)
   - Paste into Notion (Ctrl+V)

2. **Method 2 - Import:**
   - In Notion, click "+ New Page"
   - Select "Import"
   - Choose "Markdown & CSV"
   - Upload the .md file

3. **Method 3 - Markdown to Notion:**
   - Use tools like `notion-py` or `markdown-to-notion`
   - Or use Notion Web Clipper extension
